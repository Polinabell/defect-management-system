## Введение

Система управления дефектами предназначена для эффективного отслеживания, управления и решения проблем в проектах разработки программного обеспечения. Система позволяет командам работать совместно над выявлением, приоритизацией и устранением дефектов.

### Основные возможности

- **Управление проектами**: Создание и настройка проектов с различными этапами
- **Отслеживание дефектов**: Полный жизненный цикл дефектов от создания до закрытия
- **Система ролей**: Различные уровни доступа для разных типов пользователей
- **Уведомления**: Автоматические уведомления о важных событиях
- **Отчеты**: Детальная аналитика и отчеты по проектам и дефектам
- **Прикрепление файлов**: Возможность прикрепления скриншотов и документов

## Начало работы

### Вход в систему

1. Откройте веб-браузер и перейдите по адресу системы
2. Нажмите кнопку "Войти"
3. Введите ваш email и пароль
4. Нажмите "Войти в систему"

### Первый вход

При первом входе в систему рекомендуется:

1. **Обновить профиль**: Заполните дополнительную информацию о себе
2. **Настроить уведомления**: Выберите типы уведомлений, которые хотите получать
3. **Ознакомиться с проектами**: Просмотрите доступные вам проекты

### Роли пользователей

Система поддерживает следующие роли:

- **Администратор**: Полный доступ ко всем функциям системы
- **Менеджер проекта**: Управление проектами и назначение задач
- **Инженер**: Работа с назначенными дефектами
- **Наблюдатель**: Просмотр информации без возможности редактирования

## Управление проектами

### Создание проекта

1. Перейдите в раздел "Проекты"
2. Нажмите кнопку "Создать проект"
3. Заполните обязательные поля:
   - Название проекта
   - Описание
   - Дата начала и окончания
4. Нажмите "Создать"

### Настройка проекта

После создания проекта вы можете:

- **Добавить участников**: Пригласить пользователей в проект
- **Создать этапы**: Разбить проект на логические этапы
- **Настроить категории дефектов**: Определить типы проблем для проекта
- **Установить приоритеты**: Настроить уровни важности дефектов

### Управление участниками

1. Откройте проект
2. Перейдите в раздел "Участники"
3. Нажмите "Добавить участника"
4. Выберите пользователя и его роль в проекте
5. Нажмите "Добавить"

## Управление дефектами

### Создание дефекта

1. Перейдите в раздел "Дефекты"
2. Нажмите "Создать дефект"
3. Заполните форму:
   - **Заголовок**: Краткое описание проблемы
   - **Описание**: Подробное описание дефекта
   - **Проект**: Выберите соответствующий проект
   - **Категория**: Тип дефекта
   - **Приоритет**: Важность проблемы
   - **Серьезность**: Влияние на систему
4. При необходимости прикрепите файлы (скриншоты, логи)
5. Нажмите "Создать"

### Жизненный цикл дефекта

Дефект проходит следующие этапы:

1. **Новый**: Только что созданный дефект
2. **В работе**: Дефект назначен исполнителю
3. **На проверке**: Дефект исправлен, ожидает проверки
4. **Закрыт**: Дефект успешно устранен
5. **Отклонен**: Дефект не является проблемой

### Назначение дефекта

1. Откройте дефект
2. Нажмите "Назначить исполнителя"
3. Выберите пользователя из списка
4. Добавьте комментарий (опционально)
5. Нажмите "Назначить"

### Изменение статуса

1. Откройте дефект
2. Нажмите "Изменить статус"
3. Выберите новый статус
4. Добавьте комментарий о причине изменения
5. Нажмите "Сохранить"

### Добавление комментариев

1. Откройте дефект
2. Перейдите в раздел "Комментарии"
3. Введите ваш комментарий
4. Нажмите "Добавить комментарий"

### Прикрепление файлов

1. Откройте дефект
2. Перейдите в раздел "Вложения"
3. Нажмите "Прикрепить файл"
4. Выберите файл с вашего компьютера
5. Добавьте описание (опционально)
6. Нажмите "Загрузить"

**Поддерживаемые форматы файлов:**
- Изображения: JPG, PNG, GIF, BMP, WebP, SVG
- Документы: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX
- Текстовые файлы: TXT, RTF
- Архивы: ZIP, RAR

## Система уведомлений

### Типы уведомлений

Система отправляет уведомления о следующих событиях:

- **Создание дефекта**: Новый дефект в вашем проекте
- **Назначение дефекта**: Вам назначен дефект для работы
- **Изменение статуса**: Статус дефекта изменился
- **Новый комментарий**: Добавлен комментарий к дефекту
- **Системные уведомления**: Важные сообщения от администрации

### Настройка уведомлений

1. Перейдите в "Настройки" → "Уведомления"
2. Выберите типы уведомлений, которые хотите получать
3. Настройте каналы доставки:
   - Email уведомления
   - Push уведомления в браузере
   - SMS уведомления (опционально)
4. Установите время тишины (когда не получать уведомления)
5. Нажмите "Сохранить"

### Просмотр уведомлений

1. Нажмите на иконку уведомлений в верхней панели
2. Просмотрите список уведомлений
3. Нажмите на уведомление для перехода к соответствующему объекту
4. Используйте "Отметить все как прочитанные" для массовой очистки

## Отчеты и аналитика

### Доступные отчеты

1. **Отчет по дефектам проекта**: Статистика дефектов по проекту
2. **Отчет по исполнителям**: Эффективность работы команды
3. **Отчет по времени**: Анализ времени решения дефектов
4. **Отчет по категориям**: Распределение дефектов по типам

### Создание отчета

1. Перейдите в раздел "Отчеты"
2. Выберите тип отчета
3. Настройте фильтры:
   - Период времени
   - Проекты
   - Исполнители
   - Статусы дефектов
4. Нажмите "Создать отчет"

### Экспорт отчетов

1. После создания отчета нажмите "Экспорт"
2. Выберите формат:
   - PDF для печати
   - Excel для анализа
   - CSV для импорта в другие системы
3. Нажмите "Скачать"

### Дашборд

На главной странице доступен дашборд с ключевыми метриками:

- Количество активных дефектов
- Дефекты, требующие внимания
- Статистика по проектам
- Последние изменения

## Настройки профиля

### Редактирование профиля

1. Нажмите на ваше имя в верхней панели
2. Выберите "Профиль"
3. Обновите информацию:
   - Имя и фамилия
   - Email
   - Телефон
   - Фото профиля
4. Нажмите "Сохранить"

### Смена пароля

1. Перейдите в "Настройки" → "Безопасность"
2. Введите текущий пароль
3. Введите новый пароль
4. Подтвердите новый пароль
5. Нажмите "Изменить пароль"

### Настройки уведомлений

См. раздел [Система уведомлений](#система-уведомлений)

## Часто задаваемые вопросы

### Как создать новый проект?

Перейдите в раздел "Проекты" и нажмите "Создать проект". Заполните обязательные поля и нажмите "Создать".

### Как назначить дефект исполнителю?

Откройте дефект, нажмите "Назначить исполнителя", выберите пользователя и нажмите "Назначить".

### Как изменить статус дефекта?

Откройте дефект, нажмите "Изменить статус", выберите новый статус и добавьте комментарий.

### Как прикрепить файл к дефекту?

В разделе "Вложения" дефекта нажмите "Прикрепить файл" и выберите файл с вашего компьютера.

### Как настроить уведомления?

Перейдите в "Настройки" → "Уведомления" и выберите типы уведомлений, которые хотите получать.

### Как создать отчет?

Перейдите в раздел "Отчеты", выберите тип отчета, настройте фильтры и нажмите "Создать отчет".

### Как экспортировать данные?

В большинстве разделов доступна функция экспорта. Нажмите "Экспорт" и выберите нужный формат.

### Что делать, если забыл пароль?

На странице входа нажмите "Забыли пароль?" и следуйте инструкциям для восстановления.

### Как связаться с поддержкой?

Используйте форму обратной связи в разделе "Поддержка" или обратитесь к администратору системы.
